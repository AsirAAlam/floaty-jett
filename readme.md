# Floaty Jett

## To play
- Go to https://asiraalam.github.io/floaty-jett/

## Controls
- A-D   -> left-right
- W     -> jump. press again in air to double jump. hold in air to drift (jett's passive)
- S     -> press twice while on an air platform to drop down
- space -> throw knife
